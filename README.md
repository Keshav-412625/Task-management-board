# TASK MANAGEMENT BOARD

## Overview
Elite Kanban Pro is a simple Task Management web application built using React, HTML, CSS, and JavaScript. It helps users organize tasks using a Kanban board with three columns: To Do, In Progress, and Done. 0

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Move tasks between columns
- Task priority (High, Medium, Low)
- Search tasks
- Light/Dark mode
- Local Storage support
- Responsive design
- Simple and user-friendly interface 1

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- React.js

## Project Structure
```
index.html
style.css
script.js
data.json
README.md
```

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
4. Start adding and managing your tasks. 2
